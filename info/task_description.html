Not all of the elements are important. What you need to do here is to remove all the elements before a given one from the array.

For illustration, we have an array [3, 4, 5] and we need to remove all the elements that go before 3 - which is 1 and 2.

We have two possible cases here: (1) if a cutting element cannot be found, then the array shoudn't be changed; (2) if the array is empty, then it should remain empty.

Input: {% if interpreter.slug == "js-node" %}Array{% else %}Iterable{% endif %}.

Output: {% if interpreter.slug == "js-node" %}Array{% else %}Iterable{% endif %}.

Example:

{% if interpreter.slug == "js-node" %} 
removeAllBefore([1, 2, 3, 4, 5], 3) == [3, 4, 5]
removeAllBefore([1, 1, 2, 2, 3, 3], 2) == [2, 2, 3, 3]
{% else %} 
remove_all_before([1, 2, 3, 4, 5], 3) == [3, 4, 5]
remove_all_before([1, 1, 2, 2, 3, 3], 2) == [2, 2, 3, 3]
{% endif %}
